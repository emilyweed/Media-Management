##Overview
The Media Library System is a web-based application for managing media items such as books, movies, music, and games. It allows users to add, view, and search for media items using a MySQL database and an external book search API (Open Library API).

## Prerequisites
1. A local PHP server (e.g., XAMPP, WAMP, or built-in PHP server).
2. An internet connection to fetch data from the Open Library API.

## Running the File
1. Place the `1_code` folder in your server's root directory (e.g., `htdocs` for XAMPP).
2. Start your server.
3. Access the file by navigating to `http://localhost/1_code/index.php` in your browser.

##Features
- Display available media
- Search for media using Open Library API
